## 프로젝트 주요 기능
1. 카테고리 분류 및 API
2. 팝업 게시물 
3. 댓글 등록, 수정, 삭제
4. 로그인 / 로그아웃
5. 마이페이지 포트폴리오 등록 / 수정 
6. 좋아요, 팔로우, 내활동 확인

2. 페이지 리뉴얼 및 디자인 수정

4. 와이어프레임, 스토리보드 , 스케줄링표
5. React  / Styled-Components / Redux-toolkit
